# COC 部落冲突查询工具

一个基于 React 的 **Clash of Clans（部落冲突）** 数据查询与管理工具。支持查看部落信息、部落战、联赛数据以及玩家详情（兵种、英雄、装备、法术等），同时兼容夜世界数据。

## 功能特性

- **部落管理** — 添加/删除已保存的部落，查看部落详情（等级、战争胜利、连胜、成员列表等）
- **部落战** — 实时查看当前部落战的对阵情况、星级与摧毁率对比
- **联赛** — 查看联赛总览、各轮对战详情及玩家个人数据统计（总星级、进攻率、平均摧毁率）
- **玩家查询** — 查询玩家详细信息，包括：
  - 英雄等级及其装备详情（含未获得装备的灰度显示）
  - 圣水兵、黑油兵、超级兵
  - 法术、攻城器械、战宠
  - 夜世界兵种与英雄
- **数据缓存** — 部落信息本地缓存 10 分钟，减少 API 请求
- **深色主题** — 沉浸式深色 UI，适配游戏氛围
